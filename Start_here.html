Start hereÔÉÅ

All tutorials listed below may be found as Jupyter notebooks under docs/source/. We encourage running these yourself as you go through the tutorials as a means of practice. Any external data files needed such as instrument response files and numerical atmosphere data are available on Zenodo. These data files should be saved in examples/examples_modeling_tutorial/model_data/.

Overview of the available tutorials:

  • X-PSI 101 is a basic tutorial introducing the key aspects of X-PSI. In it you will build a very simple model of a single hot spot on a neutron star surface, simulate a pulse profile, and perform parameter estimation. Start here if you have never used X-PSI before!

  • Modeling is an in-depth tutorial that first covers the structure of X-PSI followed by basic usage of X-PSI. Basic usage includes constructing your first model (e.g. instrument, star, and atmosphere) and constructing a likelihood for nested sampling.

  • Instrument synergy shows how to construct a joint likelihood with data from two instruments.

  • Hot region complexity goes into more detail about multiple and especially multiple overlapping hot regions. We note that hotregion is a useful page that contains overview figures of various overlapping cases.

  • Global surface emission concerns surface emission from fields that span the full star and showcases three different options for signal integration. The first is a globally uniform temperature field, which allows for a time invariant signal integration. In the second case the temperature field is phase dependent and thus requires time dependent integration. The third case is general purpose and discretises a distant image plane instead of the stellar surface.

  • Surface radiation field tools demonstrates the usage of the default (blackbody) and an alternative (atmosphere interpolated from precomputed data) surface radiation field module to compute photon specific intensities. It also shows how to do beaming pattern and spectrum plots for the radiation fields.

  • Modeling (without statistics) is similar to Modeling but omits any statistical inference and adds various plots of signals. This is a useful tutorial if you are only interested to use X-PSI to create synthetic data.

  • Polarization is a tutorial that shows how to model polarized X-rays in X-PSI.

  • Post-processing is a simple exercise on how to use the main post-processing tools provided by X-PSI.

  • Emitting patterns 2D projection is a tutorial showing how to use the 2D projection tool for the hot emitting regions on the star‚Äôs surface.

  • Accretion disk is a tutorial showing how to set up an accretion disk and add its emission to the emission of a star.

  • Multiple imaging is a tutorial studying the effects of multiple imaging.

  • Importance sampling is a tutorial for importance sampling.

  • Module generator tutorial provides instructions on how to generate Python modules in an automated way to run X-PSI.

  • Example script and modules shows a couple of simple example scripts for pulse shape computation and nested sampling.

  • Example job contains example job scripts for computation on clusters.

  • Posterior inference using SBI is a tutorial showing how to use Simulation-Based Inference (SBI) to obtain posteriors.
